Functionality
-------------
 * All intersection tests, lighting, and light types are working.
 * Shadow testing works but is not affected by an objects transmissive properties.
 * Mirror reflection and trancparency mostly work: sometime there are weid artifacts when both are used
 * Refraction produces weird "bubble" behavior instead of expected result.
